# Requirement Defect Taxonomy

| Requirement | Defect Type | Problem | Correction |
|-------------|-------------|---------|------------|
| System should be fast | Ambiguity | "Fast" is unclear | Response time < 2 sec |
| Admin can modify data anytime | Non-verifiable | Cannot verify condition | Define specific permissions |
| Unlimited parking slots | Inconsistency | Contradicts capacity rules | Define fixed capacity |
